List out the different ways to select an item

Ways to select an item: element selector, class selector, ID selector, attribute selector, pseudo-class selector, pseudo-element selector, descendant/child/sibling combinators.

What is the difference between an ID and a Class?

ID vs Class: an ID is unique to one element, while a class can be reused on many elements.

What is the difference between a SPAN tag and a DIV tag?

SPAN vs DIV: SPAN is inline for small text-level grouping, while DIV is block‑level for larger structural grouping.

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

DIV vs HEADER/MAIN/FOOTER: DIV is generic with no meaning, while HEADER/MAIN/FOOTER are semantic tags that describe page structure.

What are the different ways to select a color?

Ways to select a color: named colors, HEX codes, RGB(), RGBA(), HSL(), HSLA(), and CSS color keywords like currentColor.
